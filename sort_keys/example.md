Write a function that returns the key sorted by length.

**Example**

For `param1 = { aaa: 3, b:1 , ddddd: 5, gg:2 }`, the output should be
`solution(param1) = [:b, :gg, :aaa, :ddddd]`

**Input/Output**

**[execution time limit] 4 seconds (rb)**

[**input**] hash param1

Guaranteed constraints:
`-1000 ≤ param1 ≤ 1000.`

[**output**] array
